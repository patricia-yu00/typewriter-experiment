### Task 3: Correct orb float zone to the glass interior

**Files:**
- Modify: `index.html` — `getJarViewBounds()` JS function

**Context:**
`getJarViewBounds()` returns `{ cx, cy, r }` — the center and radius of the circle orbs bounce inside. Currently `r = r.width * 0.38`, which puts orbs in the metallic rim area. Looking at `perspective1.png`, the dark glass interior (the red zone in the reference image) is a smaller circle centered roughly at (50%, 53%) of the image, with a radius of ~24% of the full image width.

**Step 1: Update `getJarViewBounds()`**

Find:
```js
function getJarViewBounds() {
  const bg = document.getElementById('jarViewBg');
  if (!bg) return { cx: 0, cy: 0, r: 0 };
  const r = bg.getBoundingClientRect();
  const radius = r.width * 0.38;
  return { cx: r.width / 2, cy: r.height / 2, r: radius };
}
```

Replace with:
```js
function getJarViewBounds() {
  const bg = document.getElementById('jarViewBg');
  if (!bg) return { cx: 0, cy: 0, r: 0 };
  const rect = bg.getBoundingClientRect();
  const radius = rect.width * 0.24;           // inner glass opening
  const cx     = rect.width  * 0.50;
  const cy     = rect.height * 0.54;          // slightly below center (the jar floor)
  return { cx, cy, r: radius };
}
```

    function showLetterReveal(letter) {
      document.getElementById('letterRevealDate').textContent = letter.date;
      document.getElementById('letterRevealText').textContent = letter.text;

      const reveal = document.getElementById('letterReveal');
      const paper  = document.getElementById('letterPaper');
